📄 Echo 个人时间管理与行为诊断系统 (README.md)
🌟 这是一个什么神仙项目？
你是那个每天忙忙碌碌，晚上却想不起来今天到底干了啥的人吗？
你是不是常常立了 flag 要好好学习，结果转头就在打游戏和刷视频里度过了一天，事后又后悔不已？
ECHO（Echo 个人行为环境深度诊断）就是为你准备的。
这不是一个普通的“打卡”或者“记账”软件，它是一个带有灵魂的 AI 教练。通过你对每日活动的精准记录，ECHO 会调用强大的 DeepSeek-V3 大语言模型，给你的时间分配逻辑来一次犀利、客观、拒绝空话套话的“灵魂级体检”。
做得好，AI 教练会大力赞赏；做得不好，AI 教练会毫不留情地一针见血开骂（我们称之为“灵魂体检报告”）。
核心法则：绝不偏离。

🛠️ 技术栈 
这是一个现代化的全栈系统，前后端分离，AI 服务解耦：

前端 (The Face)：基于 Vue.js 3.0 构建，集成 ECharts 5.5 实现了一个能横跨 180 天的热力大屏。颜色深度会随你当天的投入时间动态变化，一眼就能看出这半年你是在“卷”还是在“摆烂”。

后端 (The Brain)：基于 Spring Boot 3.0 构建业务逻辑。为了让你的数据不裸奔，我们集成了 Spring Security + JWT，构建了一条稳固的安全防线。

模型服务 (The Soul)：
独立分类服务：通过 Python Flask 部署了一个轻量级模型进行活动自动分类，实现真正的分类逻辑与业务解耦。

深度体检引擎：集成 DeepSeek-V3 API，专门用来撰写那些能让你当场破防或者瞬间充满动力的深度诊断报告。

存储层 (The Memory)：采用 MySQL 8.0 存储你的结构化行为数据，通过持久化记录支持长周期的趋势建模。

🚀 核心功能
行為轨迹追踪：精准记录每日事项与时长，支持跨天对比，看看你今天比昨天是不是更高效了。

180天热力大屏：直观展示过去半年的努力轨迹，颜色深度随投入时间动态变化。

AI 教練诊断：这是我们的杀手锏。根据核心学习时间占比，AI 会给你生成一篇不少于 1000 字的深度诊断报告。

>80%：高度赞赏 + 严厉压力预警。

60%-80%：极度认可的完美状态。

30%-50%：严厉敲打，警惕堕落。

<30%：极其严厉，痛骂警醒，绝不客气！

安全防护：全链路 JWT 鉴权，确保你那些不可告人的时间流水绝不泄露。

📦 快速开始 (3分钟部署指南)
1. 数据库准备 (导入家底)
安装 MySQL 8.0+。

找到项目根目录下的 db_backup.sql 文件，把它导入到你的 MySQL 中，初始化表结构。

2. 后端启动 (让大脑动起来)
使用 IDEA 打开后端项目（demo 文件夹）。

在 src/main/resources/application.properties 中，配置好你的 MySQL 连接（密码别写错）以及你的 DeepSeek API Key（就是那串以 sk- 开头的乱码）。

运行 DemoApplication.java。

3. 前端运行 (给页面点排面)
无需任何安装步骤！ 直接在浏览器里双击打开项目根目录下的 index.html 文件。

大气、全屏居中的界面就出来了。

注册一个账号，开始你的第一次“行为诊断”。
